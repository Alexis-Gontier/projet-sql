# projet-sql

## Pour lancer le projet **front** et **back** en meme temps
```bash
npm run dev
```

### Pour lancer le projet **front**
```bash
npm run frontend
```
`http://localhost:5173/`

### Pour lancer le projet **back**
```bash
npm run backend
```
`http://localhost:3000/`

Pour tester les routes avec SwaggerUI :
http://localhost:3000/api-docs/
