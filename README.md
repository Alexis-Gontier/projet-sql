# projet-sql

## Pour lancer le projet **front** et **back** en meme temps
```bash
cd projet-sql
npm run dev
```

### Pour lancer le projet **front**
```bash
cd frontend
npm run dev
```
`http://localhost:5173/`

### Pour lancer le projet **back**
```bash
cd backend
npm start
```
`http://localhost:3000/` (pas encore fait)
